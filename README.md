easymock-sample
===============

A sample project with [node-easymock](https://github.com/cyberagent/node-easymock).

## easymock install

```
$ npm install -g easymock
$ easymock
```

## project setup

```
$ npm install
$ bower install
```

## mockserver run

```
$ cd ~/YOURPATH/easymock-sample/mock-server
$ easymock
```
## app run
```
$ cd ~/YOURPATH/easymock-sample/
$ grunt server
```

## API Documentation

`http://localhost:3000/_documentation/`
